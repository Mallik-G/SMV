SMV Helper Functions

SMV provides a variety of functions to aid in the development of applications.
The provided functions come in one of three flavors.

  • Column helper functions
  • DataFrame helper functions
  • Grouped DataFrame helper functions

The Column helper functions can be use to augment the computation of columns inside a projection.
DataFrame helper functions can be used on a DataFrame directly. Whereas, Grouped helper functions can only be applied to SmvGrouped data.
